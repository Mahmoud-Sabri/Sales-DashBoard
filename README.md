📊 Sales Performance Analysis Dashboard
End-to-End Project (SQL -> Power BI)
📌 Project Overview
This project simulates a real-world business scenario where a Sales Manager needs to optimize the sales process. The goal was to transform raw data into an interactive dashboard that helps stakeholders identify pain points and growth opportunities.

🛠️ Project Stages
Requirement Gathering: Defined user stories to align the analysis with business goals.

Data Extraction: Used SQL to query and filter the required tables from the data warehouse.

Data Cleaning & Transformation: Utilized Power Query to handle nulls and ensure data type consistency.

Data Modeling: Built a robust Snowflake Schema to connect Sales, Products, Customers, and Calendar tables.

Dashboarding: Designed a user-friendly interface in Power BI using custom layouts from PowerPoint for a premium UI/UX.

🚀 Key Features
Dynamic KPIs: Real-time tracking of Sales vs. Budget.

Interactive Filter Panel: Clean UI for time and regional filtering.

Geographical Analysis: Map-based insights for global sales distribution.

Advanced DAX: Used Time Intelligence functions for Year-over-Year (YoY) growth
